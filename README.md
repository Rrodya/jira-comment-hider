# JIRA Comment Hider

Браузерное расширение для скрытия комментариев от указанных пользователей в JIRA.

## Возможности

- Скрытие всех комментариев от указанного пользователя JIRA
- Работает на всех страницах JIRA автоматически
- Сохранение настроек между сессиями
- Простой и удобный интерфейс настроек

## Установка

1.  **Склонируйте репозиторий:**
    ```bash
    git clone [https://github.com/ваш-логин/ваш-репозиторий.git]
    ```
    *Или скачайте и распакуйте ZIP-архив.*

2.  **Загрузите в браузер:**
    *   Откройте `chrome://extensions/` (или `edge://extensions/`, `opera://extensions/`).
    *   Включите **"Режим разработчика"** (переключатель вверху справа).
    *   Нажмите **"Загрузить распакованное расширение"**.
    *   Выберите папку, в которой лежит файл `manifest.json` из этого репозитория.

3.  **Готово!** Расширение установлено и активно.

### Chrome / Edge

1. Откройте Chrome и перейдите в `chrome://extensions/`
2. Включите "Режим разработчика" (Developer mode) в правом верхнем углу
3. Нажмите "Загрузить распакованное расширение" (Load unpacked)
4. Выберите папку с расширением

### Firefox

1. Откройте Firefox и перейдите в `about:debugging`
2. Нажмите "This Firefox"
3. Нажмите "Load Temporary Add-on"
4. Выберите файл `manifest.json` из папки расширения

## Использование

1. Кликните на иконку расширения в панели инструментов браузера
2. Введите полное имя пользователя JIRA (например: "JIRA Gitlab Bot")
3. Включите тогл "Включить скрытие комментариев"
4. Нажмите "Сохранить"
5. **Важно:** Если страница JIRA уже открыта, перезагрузите её (F5 или Ctrl+R / Cmd+R), чтобы изменения вступили в силу

После этого все комментарии от указанного пользователя будут автоматически скрыты на всех страницах JIRA. На новых страницах комментарии будут скрываться автоматически без перезагрузки.

## Примечание об иконках

Для работы расширения необходимы иконки. Вы можете:
- Создать PNG файлы размером 16x16, 48x48 и 128x128 пикселей
- Или использовать онлайн-генераторы иконок
- Или временно закомментировать секцию `icons` в `manifest.json`

## Структура проекта

- `manifest.json` - манифест расширения
- `popup.html` - интерфейс настроек
- `popup.js` - логика управления настройками
- `content.js` - скрипт для скрытия комментариев на странице
- `styles.css` - стили для интерфейса настроек
